Lenny-Installer

easy to use:
To see what will be executed type "curl -L stnc.cz/tool-name"
To install it "curl -L stnc.cz/tool-name | sh"

Example:
  "curl -L stnc.cz/zed | sh" Installs Zed Editor


Tools:
  [Brew.sh](https://brew.sh/)
  [Oh-My-Posh](https://ohmyposh.dev/)

Editors:
  [Zed](https://zed.dev/)
