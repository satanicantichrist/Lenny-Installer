Lenny-Installer

easy to use:
To see what will be executed type "curl -L stnc.cz/tool-name"
To install it "curl -fL stnc.cz/tool-name | sh"

To see this help just use "curl -fL stnc.cz/lenny" in your terminal

Example:
  "curl -L stnc.cz/zed.sh | sh" Installs Zed Editor


Tools:

  - brew.sh - https://brew.sh/
  - ohMyPosh.sh -  https://ohmyposh.dev/

Editors:

  - zed.sh - https://zed.dev/
